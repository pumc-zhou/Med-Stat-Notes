1 绘制基本折线图

运行 ggplot() 函数和 geom_line() 函数，并分别指定一个变量映射到 x 和 y 。

# load package
library(ggplot2)
# load dataset
BOD
  Time demand
1    1    8.3
2    2   10.3
3    3   19.0
4    4   16.0
5    5   15.6
6    7   19.8
ggplot(BOD,aes(x = Time,y = demand))+
geom_line()

# 绘制基本折线图

运行 `ggplot()` 函数和 `geom_line()` 函数，并分别指定一个变量映射到 `x` 和 `y` 。

```{r}
#| eval: true
#| echo: true
#| code-fold: false

# load package
library(ggplot2)
# load dataset
BOD

ggplot(BOD,aes(x = Time,y = demand))+
geom_line()
```
